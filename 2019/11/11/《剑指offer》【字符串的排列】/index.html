《剑指offer》【字符串的排列】

发表于 2019-11-11 | 分类于 剑指offer

题目描述

​ 输入一个字符串,按字典序打印出该字符串中字符的所有排列。例如输入字符串abc,则打印出由字符a,b,c所能排列出来的所有字符串abc,acb,bac,bca,cab和cba。

分析

字符的全排序过程可以这样描述：

  1. 找出所有可以放在第一位的字符，也就是把余下的每个字符与第一个字符交换一次。
  2. 求第一个字符外的所有字符的序列。

可以用递归来实现,最后需要对list进行排序。
